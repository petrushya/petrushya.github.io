На просторах интернета можно найти много обучающих ресурсов по созданию веб–сайтов, одним из которых является курс фронт–енд веб разработчика от MDN. Конечно, после прохождения данного курса самостоятельно, вы не получите сертификата, но приобретёте современные знания по интерфейсной веб–разработке. MDN так же целесообразно использовать и как справочник по HTML, CSS и JS.

В процессе изучения вышеназванного курса, практически с первых шагов, объясняются все возможности публикации своего творения в интернете. Там же предоставляются инструкции по размещению кода на ресурсе “GitHub Pages”, где допускается создание как простой HTML странички, так и более сложного проекта, или даже нескольких проектов, если не превышать оговоренных требований. Предлагаются различные способы переноса кода в созданный репозиторий на “GitHub Pages”. Можно просто копировать и редактировать код непосредственно на сайте, но со временем, если проснётся интерес, потре­буется изучить “Git”, для упрощения работы с репози­торием.

При выборе площадки для сайта на “GitHub Pages”, предлагается ряд полезных опций: ведение блога с “Jekyll”, использование собственного домена для сайта, путеводители по документации GitHub Pages. Для более полной функциональности “Jekyll блога” предполагается регистрация на внешнем Веб–ресурсе, таком как “Disqus”, с последующим подключением, или установка на “GitHub Pages” робота из “Staticman”, но есть у этого блога и преимущества. “Jekyll” предоставляет множество готовых бесплатных тем для оформления, публикациям не требуется модерация, идеально подходит для ведения дневника обучения и прочего. При желании, после изучения “Jekyll”, можно сконструировать полноценный сайт, соответствующий вашим запросам, больше примеров представлено в документации “Jekyll”.
